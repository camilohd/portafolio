Proceso de Trabajo

  • Levantamiento de requisitos con el cliente: Se recopila la información necesaria para el desarrollo del proyecto de la mano con el cliente.
  • Planificación: Se definen los obetivos y se planifica el curso de acción requerido para lograr los objetivos y el alcanze del proyecto. la planificación se lleva a cabo teniendo en cuenta las historias de usuario de mayor peso segun el cliente, siendo estas las primeras en desarrollarse, dejando de ultimo las menos importantes.
  • Diseños de mockups y prototipos: Se realizan los diseños iniciales de la parte visual del proyecto los cuales se discuten con el cliente.
  • Ejecución: Se da disposición a los recursos necesarios para el desarrollo del proyecto.
  • Seguimiento y Control: Se mide y se supervisa regularmente el avance, a fin de identificar las variaciónes respecto del plan de gestion del proyecto.
  • Iteraciones: Se realizan pequeñas entregas del proyecto al cliente no mayores a dos semanas, mostrando los avances realizados.
  • Cierre: Se formaliza la aceptación del proyecto con el cliente totalmente funcional y sin bugs.
